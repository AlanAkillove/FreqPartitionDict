## 一、项目概述

FreqPartitionDict 是一个基于访问频率的分区字典数据结构，采用 C++17 实现，结合哈希表（热区）和平衡树（冷区）的优势，针对倾斜访问模式（Zipf 分布）进行优化。

### 1.1 核心特性

| 特性 | 描述 | 实现状态 |
|------|------|----------|
| 双区存储 | 热区 O(1) + 冷区 O(log n) | ✅ 完成 |
| 自适应晋升 | 基于访问频率自动迁移 | ✅ 完成 |
| 频率淘汰 | 淘汰最低频率元素 | ✅ 完成 |
| 迭代器支持 | STL 兼容前向迭代器 | ✅ 新增 |
| 线程安全 | 读写锁保护并发访问 | ✅ 修复 |
| 堆优化 | O(log H) 淘汰复杂度 | ✅ 新增 |
